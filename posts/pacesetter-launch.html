Launching the Plains Pacesetter Step Challenge: Behind the Scenes

Home About Me

5 May 2025 • Calgary, AB

Last week, Plains kicked off the Pacesetter Step Challenge, a Canada-wide initiative designed to build community, support employee wellbeing, and foster a bit of friendly competition. My role? Building the data infrastructure that powers the entire program.

The challenge involves over 200 employees tracking their daily step counts, competing as teams, and staying motivated through clear, real-time feedback. To make this happen, I developed a comprehensive yet streamlined data solution, blending modern Microsoft tools to handle both the user-facing experience and backend analytics.

Technologies Used

  • Power Apps: Provided a simple, intuitive way for employees to log steps daily—right inside Teams or via the web.
  • MS Entra ID (formerly Azure AD): Leveraged existing corporate login to avoid extra passwords, simplifying the user experience and ensuring security.
  • Dataflow Gen2: Handled data ingestion from multiple sources, such as employee-submitted steps and team designation spreadsheets.
  • Microsoft Fabric SQL Server: Centralized all step data, team affiliations, and user preferences into a scalable, efficient database.
  • Power Automate: Automated routine database maintenance tasks to keep data clean and ready for reporting.
  • Power BI: Delivered real-time interactive dashboards embedded directly into SharePoint, offering immediate visibility into personal and team performance.

How I Built It

I began by creating a Power App designed specifically for easy data entry. Participants simply log their daily steps—no complicated forms or extra steps involved. By connecting this app to MS Entra ID, logging in became seamless.

The data entered flows directly into a Microsoft Fabric SQL Server instance. Before landing there, it passes through a Dataflow Gen2 pipeline, which ensures consistency and automatically resolves common data-entry errors or formatting inconsistencies.

On the analytics side, I developed a data model optimized for performance and easy integration with reporting tools. Using Power BI, I created dashboards that provide live insights into team standings and individual achievements, embedded in SharePoint for universal accessibility.

To maintain data integrity, I employed Power Automate to automate cleanup tasks and trigger routine data processes, eliminating the need for manual intervention.

Lessons Learned

  • User experience matters: Making the app as straightforward as possible significantly improved user satisfaction.
  • Real-time insights enhance engagement: When participants see immediate results, motivation grows stronger.
  • Automate whenever possible: Anticipating and automating routine tasks early on saved significant time down the road.

TL;DR: Built the data backbone for Plains' company-wide step challenge, using Power Apps, MS Fabric SQL, Power Automate, and Power BI, resulting in a scalable and user-friendly solution.
